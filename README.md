# CrazyC

《Crazy Countdown》

偶爾想打發時間玩玩卡牌類遊戲，但是又不想花太多心力在培養卡片上面嗎？
想嘗試有別於傳統撲克牌的策略卡片遊戲嗎？
《Crazy Countdown》是你的好選擇！

遊戲特色
1.規則類似UNO及Crazy Eight簡單上手！ 
2.不同的效果卡可以進行攻擊對手或是防禦攻擊！
3.採用積分的方式來決勝負！
4.緊湊的遊戲步調 最速決定勝負！！

We hope the project will help you!
